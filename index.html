JS Cheat-Sheet

Basic Data Types:

JavaScript has 4 main basic data types

  • Strings: "Hello World"
  • Numbers: 1.1,2,-3,4000
  • Booleans: True or False
  • Undefined: noValue

    • // console.log("Jose") -> Will log a string

    // console.log(3) -> Will log a number

    Variables

    Variables are like boxes that store information that can be changed at any point

    They have a key and a value

    // var myName = "Jose";

    // var dogBreed = "Terrier";

    // var apples_number = 3;

    Arrays

    Arrays store multiple values in a single variable. The index of the array (first value) starts with the number of 0

    The array cars=['Volvo', 'Porsche', 'BMW'] has 3 values but since the index stars at 0, the position of the values are:
    0 = 'Volvo', 1 = 'Porsche' and 2 = 'BMW'.

    // Arrays are written like this ->
    fruits = ['apples', 'oranges', 'bananas'];

    Logic

    Logical operators are used for comparisons and to test whether a statement meets certain conditions

    // Equal to: 2 == 2 -> true

    // Equal type and value: '2' === 2 -> false

    // Not Equal: 2 != 3 -> true

    // Greater than: 2 > 1 -> true

    // Less than: 2 < 3 -> true

    Find more at: W3Schools

    Functions

    Functions are like recipes that tell the computer what to do, step-by-step.

    // This are written like so:
    function doThis(parameter) {
    /* code to be executed */
    }

    To call a function, just write the name and give it a parameter, if needed

    // doThis("parameter");
